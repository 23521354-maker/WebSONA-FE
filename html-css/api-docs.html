API Documentation

Tài liệu API cho hệ thống quản lý sản phẩm Sona

Base URL: http://localhost:3000/api
Products Authentication Orders

Products API

GET /products
Lấy danh sách tất cả sản phẩm

Response Example:

[
  {
    "ID_SP": 1,
    "TenSP": "Intel core i5 14600KF",
    "GiaBan": 5600000,
    "HinhAnhChinh": "img/Sanpham/intel-core-i5-14600kf.webp",
    "TenDM": "CPU",
    "TenHang": "Intel"
  }
]
GET /products/:id
Lấy chi tiết một sản phẩm theo ID

Parameters:

id ID của sản phẩm (path parameter)

Response Example:

{
  "ID_SP": 1,
  "TenSP": "Intel core i5 14600KF",
  "GiaBan": 5600000,
  "HinhAnhChinh": "img/Sanpham/intel-core-i5-14600kf.webp",
  "ID_DM": 1,
  "TenDM": "CPU",
  "TenHang": "Intel",
  "CPU": "Intel Core i5-14600KF"
}
GET /products/flashsale
Lấy danh sách sản phẩm Flash Sale
GET /products/bestsellers
Lấy danh sách sản phẩm bán chạy nhất
GET /products/accessories
Lấy danh sách phụ kiện
GET /products/search
Tìm kiếm sản phẩm theo từ khóa

Query Parameters:

q Từ khóa tìm kiếm
GET /products/category/:categoryId
Lấy sản phẩm theo danh mục

Parameters:

categoryId ID của danh mục

Authentication API

POST /auth/register
Đăng ký tài khoản mới

Request Body:

hoTen Họ tên người dùng
email Email
matKhau Mật khẩu
soDienThoai Số điện thoại
POST /auth/login
Đăng nhập hệ thống

Request Body:

email Email đăng nhập
matKhau Mật khẩu

Orders API

POST /orders
Tạo đơn hàng mới

Request Body:

items Mảng các sản phẩm trong đơn hàng
diaChi Địa chỉ giao hàng
GET /orders/:id
Lấy chi tiết đơn hàng

Parameters:

id ID của đơn hàng
